# Swiss-Hoopps
<Basketball game>

   *Tutorial*
   
  How to play: Drag the ball with the mouse/touch and release to shoot. The power meter shows the power. The dotted line predicts the trajectory when pulling.

Keyboard controls: R = reset, Space = retake ball, M = mute, F = fullscreen.

Mode: "Mode: Casual" button changes to Arcade (slightly harder, higher scores).

Time Mode: the "Start 60s" button starts a 60-second time session; once the time runs out you can save your score.

Leaderboard: save scores with the "Save Scores" button, saved in localStorage (top 50 saved, UI displays top 10).

Sound: created via Web Audio; click/press button to activate music in some browsers(autoplay policy).
